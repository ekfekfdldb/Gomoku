# Gomoku (오목)

C# Windows Forms 기반으로 제작된 오목 게임 프로젝트입니다.  
직관적인 GUI 환경에서 오목을 즐길 수 있고, **로그인 시스템**, **금수(금지수) 판정**, **AI 대전**,  
**턴 타이머** 등 여러 기능을 통해 오목 규칙을 학습·연습할 수 있습니다.

---

## 📝 프로젝트 소개 (Introduction)

이 프로젝트는 객체 지향 프로그래밍(OOP)을 바탕으로  
C#과 .NET Framework(Windows Forms)를 사용해 구현되었습니다.

- 게임 규칙과 보드 상태는 **로직 클래스(OmokRules, OmokAI)** 에서 처리하고  
- 화면 렌더링과 버튼/이벤트 등은 **폼 클래스(UnifiedMainForm)** 에서 처리하는 식으로  
  **UI와 게임 로직이 분리**되어 있습니다.

플레이어 간 로컬 대전뿐만 아니라, AI와의 대전도 지원합니다.

## 📜 게임 규칙 (Game Rules)

### 1. 기본 규칙

- **보드 크기**: 19 × 19  
- **선공**: 흑(Black)이 먼저 둡니다.  
- **승리 조건**  
  - 가로 / 세로 / 대각선 방향으로 같은 색 돌 **5개가 연속**으로 놓이면 승리  
  - 구현된 규칙에 따라, 6목 이상(장목)은 금수로 처리될 수 있습니다.

---

### 2. 진행 방식

- 흑과 백이 **한 번에 한 수씩 번갈아 착수**합니다.
- 이미 돌이 놓인 칸에는 착수할 수 없습니다.
- 매 수를 둘 때마다 `OmokRules.cs`에서 다음을 검사합니다.
  - **5목 달성 여부**
  - **금수 여부**

판정 결과에 따라 즉시 게임이 계속 진행되거나 종료됩니다.

---

### 3. 금지수 (흑 전용)

`OmokRules.cs`에 구현된 룰에 따라, 흑에게 다음과 같은 수들이 금수로 처리됩니다.

- **3×3 금수**: 두 개 이상의 열린 3을 동시에 만드는 수  
- **4×4 금수**: 두 개 이상의 열린 4를 동시에 만드는 수  
- **장목 금수**: 정확히 5목이 아닌, 6목 이상의 연속된 돌을 만드는 수  

금수 좌표는 보드 위에 **빨간 X 표식**으로 표시되며,  
흑이 그 위치에 두려고 하면 **착수가 무효 처리**됩니다.

## ✨ 주요 기능 (Features)

- **로그인 / 회원가입 시스템**
  - 간단한 메모리 기반 계정 관리
  - 닉네임, 가입일 조회(「내 정보」 메뉴)

- **정식 오목 규칙 + 금수 판정**
  - 5목 체크, 3×3 / 4×4 / 장목 금수 구현
  - 흑 차례일 때만 금수 표시

- **AI 대전 모드 (`OmokAI.cs`)**
  - 사람 vs AI 대결
  - 보드 평가 점수 기반의 간단한 수읽기 로직

- **턴 타이머 & 프로그레스바**
  - 각 플레이어에게 **30초 제한 시간**
  - 남은 시간을 **부드러운 애니메이션** 형태의 막대로 표시
  - 시간 초과 시, 현재 플레이어가 자동으로 한 수 착수 후 턴이 넘어감

- **UI / 연출**
  - 인트로 로고 페이드인, 로딩 애니메이션
  - 메인 메뉴 전용 배경 이미지
  - 커스텀 버튼, 메뉴, 보드 렌더링

## 🚀 실행 방법 (How to Run)

1. 레포지토리를 클론합니다.

   ```bash
   git clone https://github.com/ekfekfdldb/Gomoku.git
	```
2. Visual Studio에서 Game/Game.csproj 를 엽니다.

3. 시작 프로젝트를 Game으로 설정합니다.

4. 실행(F5)하면 로그인 → 메인 메뉴 → 게임 화면 순서로 진행됩니다.
